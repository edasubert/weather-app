# Weather app

This is an exercise in vibe coding. For a while I wanted a weather app that would compare the forecast to what the weather has been yesterday. So I finally made one, also there is no tracking, no cookie consent, no bs like that (well I don't know what open-meteo does, but this app does not).

It has been made entirely with Claude code, with only minor changes from me.

## How I use it

I like to use [Firefox Focus](https://www.firefox.com/en-US/mobile/focus/) for my mobile browsing, but that makes repeated visits to websites with cookie consents and settings in cookies real annoying. That is why this app is designed to store all settings in the url. So I created a url with my home location and saved it on my home screen. Now I can visit it directly anytime with everything set up the way I want it.
